# envboy
Arduino environment monitor with display and get data as json over HTTP.

## hardware
* ESP8266
* BME280
* SSD1306
* TSL2561 (Optional)
* LPS22HB (Optional)

## how to use
* write sketch
* boot (as setup mode)
* connect to wifi (SSID is eb3*******)
* access http://192.168.4.1/ and setting.
* reset
* boot (as normal mode)
* enjoy!
